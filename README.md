# Job-Application-Tracker
A clean, beginner-friendly, and aesthetic web app to track job applications — built using HTML, CSS, and JavaScript.
# About the project
You can track:
*Company name
*Job role / position
*Application status
*Date applied
*Notes
Everything is stored in your browser using Local Storage — no database needed!
# Features
✔️ Add new job applications
✔️ Update application status
✔️ Delete applications
✔️ Data saved automatically (even after refresh)
✔️ Clean pastel UI with a professional aesthetic
✔️ Fully responsive design
✔️ Beginner-friendly code structure
# Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
Local Storage API
GitHub Pages (for deployment)
